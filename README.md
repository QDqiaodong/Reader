# Reader
#### 高仿多看阅读器apk，支持PDF解析.文档支持搜索，高亮文字，墨迹等功能

１．目录（解析文件目录，并且可以实现跳转）

２．搜索功能

３．浏览模式（模仿多看PDF浏览功能，单页浏览/滚动浏览）

４．复制

５．高亮

６．下划线

７．删除线

８．墨迹

９．支持页面缩放

demo下载：[Reader.apk](https://github.com/QDqiaodong/Reader/blob/master/Demo/reader.apk)
